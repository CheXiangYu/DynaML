    Differential Operator

    An abstract idea of a differential operator, it is expressed as a DataPipe which takes a function/tensorflow layer and returns another function/tensorflow layer.

    Differential operators are applied on some space of functions, in DynaML (and TensorFlow), differentiable functions are represented as computational layers.

    I

    Input domain of the underlying function space.

    J

    Output domain of the underlying function space.
