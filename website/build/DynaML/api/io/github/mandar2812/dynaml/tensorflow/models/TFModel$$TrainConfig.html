TrainConfig 

case class TrainConfig[In](summaryDir: Path, data_processing: Ops[In] = TFModel.data_ops(10000, 16, 10), optimizer: Optimizer = tf.train.Adam(0.01f), stopCriteria: StopCriteria = ..., trainHooks: Option[Set[Hook]] = None) extends Product with Serializable

A training configuration, contains information on optimization method, convergence test etc.

summaryDir

A filesystem path of type ammonite.ops.Path, which determines where the intermediate model parameters/checkpoints will be written.

optimizer

The optimization algorithm implementation.

stopCriteria

The stopping criteria for training, for examples see Learn.max_iter_stop, Learn.abs_loss_change_stop and Learn.rel_loss_change_stop

trainHooks

A set of training hooks to run, training hooks perform activities such as saving the current TensorFlow graph, saving the loss, step rate, summaries (audio, video, tensor) etc. See the _train_hooks() method for setting some default training hooks.
