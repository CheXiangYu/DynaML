In

The type representing symbolic tensors of the input patterns, e.g. Output, (Output, Output), Seq[Output] etc.

Out

The type representing symbolic tensors of the target patterns, e.g. Output, (Output, Output), Seq[Output] etc.

ArchOut

The type of the symbolic tensor returned by the neural architecture, e.g. Output, (Output, Output), Seq[Output]

IT

The type representing input tensors, e.g. Tensor, (Tensor, Tensor), Seq[Tensor] etc.

ID

The input pattern's TensorFlow data type, e.g. FLOAT64, (FLOAT64, FLOAT64), etc.

IS

The type of the input pattern's shape, e.g. Shape, (Shape, Shape), Seq[Shape]

TT

The type representing target/label tensors, e.g. Tensor, (Tensor, Tensor), Seq[Tensor] etc.

TD

The target pattern's TensorFlow data type, e.g. FLOAT64, (FLOAT64, FLOAT64), etc.

TS

The type of the target pattern's shape, e.g. Shape, (Shape, Shape), Seq[Shape]
