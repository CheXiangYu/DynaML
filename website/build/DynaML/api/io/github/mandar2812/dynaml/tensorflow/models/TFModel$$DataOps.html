DataOps 

case class DataOps[I](shuffleBuffer: Int = 10000, batchSize: Int = 16, prefetchSize: Int = 16, repeat: Int = -1, seed: Option[Int] = None, custom_data_pipe: Option[DataPipe[Dataset[I], Dataset[I]]] = None) extends Product with Serializable

Defines data operations to be performed using TensorFlow data API.

shuffleBuffer

The size of the shuffle buffer, set to zero if data shuffling is not needed.

batchSize

Size of the mini batch.

prefetchSize

Number of elements to prefetch.

repeat

Decides how many times the tensorflow data set is to be repeated. Defaults to -1 which repeates it infinitely.
