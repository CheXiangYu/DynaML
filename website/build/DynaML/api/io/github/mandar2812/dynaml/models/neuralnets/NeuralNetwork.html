NeuralNetwork 

trait NeuralNetwork[G, P, Input, Output, T <: NeuralGraph[P, Input, Output], Pattern] extends ParameterizedLearner[G, T, DenseVector[Double], DenseVector[Double], Stream[Pattern]]

Top level trait defining properties of a neural network.

G

The type of the data structure containing the training data set.

P

The underlying base graph from which the NeuralGraph object is constructed.

T

A subclass of the NeuralGraph object with P as the base graph

Pattern

The type of an individual data pattern

Annotations
