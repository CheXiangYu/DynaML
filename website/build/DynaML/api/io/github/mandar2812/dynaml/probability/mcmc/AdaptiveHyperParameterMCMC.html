AdaptiveHyperParameterMCMC 

class AdaptiveHyperParameterMCMC[Model <: GloballyOptimizable, Distr <: ContinuousDistr[Double]] extends RandomVariable[Map[String, Double]]

Adaptive Metropolis Sampler

Adaptive Markov Chain Monte Carlo algorithm, equipped with reversible Metropolis Hastings updates.

The proposal distribution is a multivariate gaussian whose variance is adjusted every iteration.

Model

Any type which implements the GloballyOptimizable trait i.e. the GloballyOptimizable.energy() method.

Distr

A breeze distribution type for hyper-parameter priors.
